## Minnesota Supercomputing Institute Help Desk

We are the user facing support team for user of MSI systems. Our primary goal is to help users that find themselves stranded get back on the road by troubleshooting system and software issues while working in a shared high performance compute environment.

- [Support Tools](https://github.com/umn-msi-helpdesk/support_tools) Collection of tooling utilized by Help Desk staff for handling specific requests.
- [User Documentation](https://github.com/umn-msi-helpdesk/user-documentation) Github pages site for users of MSI systems.
